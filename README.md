# Squeeze

A browser-based image compression tool that helps users reduce image file sizes quickly without uploading their files to a server.

Squeeze is designed as a simple privacy-friendly utility: users can drag and drop images, choose compression settings, preview the results, and download optimized files directly from the browser.

## Live Site

[Squeeze](https://joshelieson.github.io/Squeeze-Image-Compressor/)

## Why I Built It

I wanted to build a small, polished utility that solves a common problem: reducing image sizes for websites, portfolios, applications, and uploads. The goal was to make the experience fast, clean, and private by handling compression directly in the browser.

## Features

- Drag-and-drop image upload
- Client-side image compression
- Adjustable compression settings
- Output format selection
- Before/after file size comparison
- Image preview before downloading
- Fast, responsive interface
- Privacy-friendly browser-only workflow

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Image Processing:** Browser canvas / client-side compression tools
- **Deployment:** Vercel

## Engineering Highlights

- Built a fully client-side image processing workflow
- Used browser APIs to preview, resize, compress, and export images
- Designed a clean utility-focused interface with minimal friction
- Avoided server-side storage to keep user files private
- Optimized the app for fast interaction and immediate feedback

## What I Learned

This project strengthened my frontend engineering skills around browser APIs, file handling, image processing, responsive UI design, and building small tools that feel polished and useful.
